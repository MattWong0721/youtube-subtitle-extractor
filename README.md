# YouTube Subtitle Extractor

Minimal web app to extract subtitles from any YouTube video. Paste a link, choose a duration, download as SRT / CSV / Excel.

## Setup

```bash
pip install -r requirements.txt
python app.py
```

Open http://localhost:5555

## Usage

1. Paste a YouTube URL
2. Optionally set a duration limit (in minutes)
3. Click "Extract Subtitles"
4. Download in SRT, CSV, Excel, or all at once as a ZIP
